# wp-env

```sh
export DOCKER_HOST=unix://$XDG_RUNTIME_DIR/docker.sock
dockerd-rootless
docker run hello-world

dockerd-rootless & npm run wp-env start

npm run wp-env stop
npm run wp-env clean
docker ps
```
